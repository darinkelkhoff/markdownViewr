markdownViewr 1.4.0

Released June 26, 2026

Native macOS toolbar customization

  • Added native macOS toolbar customization for document windows.

View settings now apply per window

  • TOC, TOC depth, Markdown Source, zoom, and active theme are now per-window settings.
  • Added Settings options to define default settings for new windows.

Menu and shortcut improvements

  • Cleaned up the File menu by removing New, Save, and Revert, and adding Open in External Editor.
  • Added View menu commands for TOC, Markdown Source, and TOC depth.
  • Added Cmd-T to toggle TOC, Cmd-U to toggle Markdown Source, and Cmd-E to open in the external editor.

Window and settings polish

  • Improved default window sizing and side pane widths.
  • Added per-file window size restoration.
  • Added the ability to reorder external editors.
