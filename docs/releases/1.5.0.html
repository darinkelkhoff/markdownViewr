markdownViewr 1.5.0

Released July 23, 2026

Native printing

  • Added native macOS printing for rendered Markdown documents.
  • Added a Printing tab in Settings for print theme, images, backgrounds, borders, page padding, zoom, and content width.
  • Added a built-in Clean Printing theme for predictable print output.

Toolbar improvements

  • Added an optional Print toolbar item that can be added from Customize Toolbar.
  • Replaced the custom small-space toolbar item with AppKit's native repeatable space item.

Window and navigation polish

  • Updated table-of-contents highlighting to follow the mouse cursor instead of just the top-of-window scroll position.
  • Improved tabbed document window behavior by avoiding frame autosave on tab groups.
