markdownViewr 1.3.0

Released June 19, 2026

New: Markdown source view

  • Added a side-by-side view of raw Markdown source, with scrolling synchronized with the rendered view.
  • Find works across the rendered document and source pane.
  • Heading collapse state is mirrored between the rendered document and source pane.

Table of Contents updates

  • Made the table of contents pane resizable.
  • Made TOC width, visibility, and heading depth persistent.
  • Added optional TOC depth markers (bullets) and smoother active-heading tracking while scrolling long documents.

Customization and rendering polish

  • Improved scrollbar appearance by styling them to match the active theme.
  • Expanded the CSS help with custom CSS examples, document structure notes, and scrollbar styling selectors.
  • Fixed user-created theme overrides so they reliably take precedence over built-in themes.
